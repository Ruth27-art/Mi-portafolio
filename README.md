# 🛡️ Portafolio de Ciberseguridad - Ruth Cáceres

Bienvenido al código fuente de mi portafolio profesional. 

Este proyecto está construido con una arquitectura moderna utilizando:
- **React** (Biblioteca de interfaces de usuario)
- **Vite** (Herramienta de compilación ultrarrápida)
- **TypeScript** (JavaScript con tipado estricto)
- **CSS3** (Diseño responsivo con Glassmorphism)

El despliegue está completamente automatizado utilizando **GitHub Actions** hacia **GitHub Pages**.
